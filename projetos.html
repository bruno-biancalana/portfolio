Clone Instagram

  Este projeto consiste na recriação da página de login do Instagram, com HTML5 e CSS3. A parte mais dificil neste caso é a estilização e alocação dos elementos, pois o design da página tem um conceito bem moderno. A parte editada em HTML contem cerca de 41 linhas de código, enquanto que a parte em CSS possui mais de 150 linhas. Este projeto esta disponivel no meu repositório do GitHub.

Site para P.I

Este site, também disponivel no meu Git Hub, foi desenvolvido para o Projeto Integrado do curso de ciencias da computação da Faculdade das Américas, na qual sou discente. O site serve para apresentar um produto, o Hidrometro monitorador de consumo com Arduino. O site possui a API do whatsapp e do Google Maps (O mapa esta fora do ar pois a API key do Maps foi retirada, para evitar cobranças). Desenvolvido com auxilio do Bootstrap, possui 3 páginas e um menu navegável. O site está no ar, hospedado em um servidor gratuito, disponivel através do link: http://pifam4sem.infinityfreeapp.com/index.html?i=1

Estacionamento automatizado de baixo custo

Projeto de conclusão de curso do Técnico em Eletronica do Senai. O projeto tinha por objetivo reduzir os custos de manutenção/mão de obra de um estacionamento, automatizando-o de forma barata. Com um microcontrolador da familia PIC16F programado em linguagem C, led's infravermelhos e fototransistores, foi montada uma maquete para materializar o projeto. O artigo deste protótipo esta disponivel aqui.
